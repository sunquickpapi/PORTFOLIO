CookKing Application

A specialized mobile application designed to simplify healthy cooking and recipe management for home cooks.

My Role: Mobile App Developer

The Challenge

Home cooks often struggle with two main issues: deciding "what to eat" from a variety of cuisines (Malay, Western, Thai) and executing complex recipes without errors. Following static text recipes can lead to timing mistakes or incorrect portion sizes, especially for intricate local dishes like Penang Laksa. There was a need for an interactive digital assistant that not only lists ingredients but actively guides the user through the cooking process.

The Solution

Developed Cook King (CK), a native iOS application built using SwiftUI that serves as an interactive kitchen companion. The application provides Smart Cooking Assistance through an integrated "Smart Timer" that allows users to track cooking durations for specific steps directly within the workflow. To ensure perfect results, I implemented Dynamic Portion Control with a "Choose Your Pax" selector that automatically calculates and adjusts ingredient quantities based on the number of people eating.

Beyond basic guidance, the app emphasizes Personalization by allowing users to jot down recipe tweaks via a "Write Your Notes" feature and set "Cook Later" reminders. Navigating the recipe database is effortless thanks to a robust Discovery Engine, featuring a categorized search system across Healthy, Western, Malay, and Thai cuisines to help users quickly find the perfect dish for any preference.

🏆

Results & Impact

The app successfully bridged the gap between recipe discovery and execution. The built-in countdown timer significantly Reduced Cooking Errors by lowering the risk of overcooking time-sensitive steps, while the interactive "Step-by-Step" guide maintained high levels of User Engagement compared to static blogs. Additionally, the "Cook Later" feature provided immense Convenience, helping users plan their meals and grocery runs well in advance.

💡

Key Learnings

This project was a deep dive into SwiftUI state management. I learned how to use @State and @Binding to pass data between views—such as passing a selected dish to the detail view—and how to manage precise timer logic using Timer.publish. It also highlighted the importance of a clean UX/UI, utilizing intuitive icons and card layouts to make the app feel friendly, accessible, and ready for the kitchen environment.
